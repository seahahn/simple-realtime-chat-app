# App for Sailrs

- [Go to Deployed Web Site](https://simple-realtime-chat-app-six.vercel.app/)

## Features

### User Authentication Implementation

- Sign Up
- Sign In
- Sign Out
- Reset Password
- User Profile Update
- User Account Delete

## How to Run

From your terminal:

```sh
npm run dev
```

This starts your app in development mode, rebuilding assets on file changes.
